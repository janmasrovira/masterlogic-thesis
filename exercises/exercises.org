* Set 1
  Show the following:
  # 1. $IL⊬R$
  1. $IL⊬A▷B→¬(A▷¬C)▷B∧□C$.

     We need to find a model $M$ and a world $w$ such that $M,w⊩A▷B$ but
     $M,w⊮¬(A▷¬C)▷B∧□C$
  2. $IL⊬A▷B→A▷(B∧□¬A)$

     We need to find a model $M$ and a world $w$ such that $M,w⊩A▷B$ but
     $M,w⊮A▷(B∧□¬A)$.
     1. So for all $wRu$ such that $M,u⊩A$ there is a $v$ such that $uS_wv$ and
        $M,v⊩B$.
     2. There exists a world $y$ where $wRy$ and there is no $v$ such that
        $yS_wv$ and $M,v⊩B∧□¬A$.
  3. $IL⊬A▷B→♢A∧□C▷B∧□C$
  4. $IL⊢A∧□¬A▷A$
  5. $IL⊢A▷A∧□¬A$
  6. $IL⊢⊤▷□⊥$
